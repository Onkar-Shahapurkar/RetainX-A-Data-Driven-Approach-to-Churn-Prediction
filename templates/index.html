    Churn Insights

    Key findings and actionable insights from our customer churn analysis.

    Contract Length Impact

    Month-to-month contracts have a 42% churn rate compared to just 8% for two-year contracts.

    Recommendation: Incentivize longer contract commitments with special offers and discounts.

    Security Services

    Customers without online security and tech support are 3x more likely to churn.

    Recommendation: Offer free trials of security services to high-risk customers.

    Pricing Sensitivity

    Higher monthly charges correlate strongly with increased churn, especially for new customers.

    Recommendation: Create tiered pricing plans with gradual increases for the first year.

    About This Project

    This customer churn prediction tool was developed using machine learning techniques to help telecom companies identify and retain at-risk customers. The model was trained on historical customer data and uses gradient boosting algorithms to predict churn probability.

    Model Details

    • Gradient Boosting Classifier
    • SMOTEENN for imbalanced data handling
    • Feature selection with SelectKBest
    • Hyperparameter tuning via RandomizedSearchCV
    • Model accuracy: 84.5%

    Technologies Used

    • Python & Scikit-learn for ML model
    • Flask for backend API
    • HTML, CSS & JavaScript for frontend
    • Chart.js for data visualization
    • Tailwind CSS for styling
